📌 Global Airbnb Performance Dashboard


📊 Problem Statement

Analyze Airbnb data to uncover trends in listings, pricing, customer behavior, and market performance across cities.



🎯 Objective

Build an interactive dashboard to generate actionable insights for pricing strategy, seasonal demand, and customer engagement.



🗂️ Dataset

Source: Maven Airbnb Dataset
Size: 270K+ listings, 5M+ reviews
Features: City, Price, Room Type, Reviews, Dates



🚀 Business Impact

Helps optimize pricing strategies
Identifies high-demand markets
Improves understanding of customer behavior
Supports strategic decision-making



🚀 “Business Recommendation” Section

Suggest dynamic pricing for peak months (Apr–Aug)
Encourage hosts in low-performing cities to adopt pricing strategies from top cities
Improve repeat engagement (since 96% users review once) via loyalty programs



🛠️ Tech Stack

Power BI
Excel
DAX



🔍 Methodology

Data Cleaning & Preprocessing
Exploratory Data Analysis
Feature Understanding
KPI Creation using DAX
Dashboard Development



📈 Key Insights

Listings peaked in 2015 and declined post-regulations
Paris, NYC, Sydney dominate listings & reviews
Hotel rooms (~$800) are most expensive
96% users review only once
Peak demand: April–August
66% hosts verified → strong trust indicator



⚠️ Challenges Faced

Handling large-scale data efficiently
Missing and inconsistent data
Skewed review distribution
Designing intuitive yet insightful dashboards
Writing accurate DAX calculations
